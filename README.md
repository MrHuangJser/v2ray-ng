# V2rayNg

A modern v2ray implementation using by electron and angular

## release

for some reason there won't provide release file, you can use it by

```zsh
git clone https://github.com/hwRepoOrg/v2ray-ng.git
cd v2ray-ng
```

then

### for mac

```zsh
yarn && yarn build:mac
```

### for win

```zsh
yarn build:win
```
